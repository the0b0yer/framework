Breakpoint Responsive

Il est possible de créer des colonnes responsives et de leur indiquer la taille qu'elles vont prendre en fonction de la résolution de l'écran.

Pour utiliser les class responsives, le contenu doit être dans un conteneur ayant la class container et celui-ci doit contenir un div comme child.

Le code devrait ressembler à ceci : class="container" > div > class="col...".

Ces tailles peuvent être modifiées dans le code en modifiant les variables $media-sm, $media-md et $media-lg.

Le nombre de colonnes actuellement est de 12 mais il peut être modifié grâce à la variable $nb_columns.

Le margin entre les colonnes est actuellement de 20px mais il peut être modifié grâce à la variable $margin-grid.

Taille des breakpoints

  • Taille par défaut qu'importe la résolution de l'écran :
    • class : col
    • class : col-"1 à 12"
  • Taille small (sm) :
    • Taille max 480px
    • Class : col-sm-"1 à 12"
  • Taille medium (md) :
    • Taille max 660px
    • Class : col-md-"1 à 12"
  • Taille large (lg) :
    • Taille max 1160px
    • Class : col-lg-"1 à 12"
  • Taille extra-large (xl) :
    • Taille à partir de 1161px
    • Class : col-xl-"1 à 12"

Margin & Padding

Il est possible de rajouter rapidement des margins et paddings à n'importe quel élément.

Soit localisé : top, bottom, left, right, en x (left et right), en y (top et bottom) ou en auto.

On peut augmenter le nombre de class en modifiant dans le code les variables $nb_margin pour les margins et $nb_padding pour les paddings. Il est aussi possible de modfier l'unité das laquelle ils sont exprimés grâce à la variable $unit_margin et $unit_padding.

Margin

  • Margin auto :
    • class : m-auto
    • class : mx-auto
    • class : my-auto
  • Margin :
    • class : m-"0 à 12"
    • class : mx-"0 à 12"
    • class : my-"0 à 12"
  • Margin localisé (top, bottom, left, right) :
    • class top : mt-"0 à 12"
    • class bottom : mb-"0 à 12"
    • class left : ml-"0 à 12"
    • class right : mr-"0 à 12"

Padding

  • Padding auto :
    • class : p-auto
    • class : px-auto
    • class : py-auto
  • Padding :
    • class : p-"0 à 12"
    • class : px-"0 à 12"
    • class : py-"0 à 12"
  • Padding localisé (top, bottom, left, right) :
    • class top : pt-"0 à 12"
    • class bottom : pb-"0 à 12"
    • class left : pl-"0 à 12"
    • class right : pr-"0 à 12"

Couleurs

Il est possible de rajouter rapidement des couleurs dans la variable $all-colors sous forme de class_de_votre_couleur: le_code_de_votre_couleur,.

Cela va permettre de générer automatiquement les class de la couleur du texte et du background ajouté.

Il est possible d'ajouter à ces class une opacité.

Couleurs

  • Liste des couleurs actuelles :
    • class : white
    • class : black
    • class : blue
    • class : red
  • Couleur des textes :
    • class : txt-"Nom_de_la_couleur"
  • Couleur des textes avec opacité :
    • class : txt-"Nom_de_la_couleur"-"0 à 10"
  • Couleur des backgrounds :
    • class : bg-"Nom_de_la_couleur"
  • Couleur des backgrounds avec opacité :
    • class : bg-"Nom_de_la_couleur"-"0 à 10"

Bordures

Il est possible de faire rapidement bordures en fonction des couleurs et de rajouter si besoin, une taille.

La bordure peut aussi être localisée : top, bottom, left, right, en x (left et right) ou en y (top et bottom).

Le nombre de class pour l'épaisseur des bordures peut être modifié grâce à la variable $nb_border_radius.

On peut aussi modifier l'unité exprimée de la bordure via la variable à la variable $unit_border.

Bordures

  • Bordure :
    • class : bc-"Nom_de_la_couleur"
    • class : bcx-"Nom_de_la_couleur"
    • class : bcy-"Nom_de_la_couleur"
  • Bordure localisé (top, bottom, left, right) :
    • class top : bct-"Nom_de_la_couleur"
    • class bottom : bcb-"Nom_de_la_couleur"
    • class left : bcl-"Nom_de_la_couleur"
    • class right : bcr-"Nom_de_la_couleur"
  • Bordure avec épaisseur :
    • class : bc-"Nom_de_la_couleur"-"0 à 10"
    • class : bcx-"Nom_de_la_couleur"-"0 à 10"
    • class : bcy-"Nom_de_la_couleur"-"0 à 10"
  • Bordure localisé avec épaisseur (top, bottom, left, right) :
    • class top : bct-"Nom_de_la_couleur"-"0 à 10"
    • class bottom : bcb-"Nom_de_la_couleur"-"0 à 10"
    • class left : bcl-"Nom_de_la_couleur"-"0 à 10"
    • class right : bcr-"Nom_de_la_couleur"-"0 à 10"
